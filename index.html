Path

It is the exact location of an image or a source in a device.
they are two types:

  1. Absolute file path
    • This file path is used for access the long distance file.
      Suppose we are working at a folder in D drive. absolute file path can be used for accessing a file in c drive.
    • eg: C:/downloads/image/new.png
  2. Relative file path
    • This file path is used for access a nearby source or an image such as in in the same folder that we are working with.
    • Single dot can be used for an image in the same folder.
      • eg:"./image.png"
    • Double dot can be used for access a source from form one step back folder
      • eg:"../image/flower.png"
